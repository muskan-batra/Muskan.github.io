Bluetooth-Chat app

For communication purposes we have many ways, through the internet, cellular or other ways. Bluetooth is a common and cheap technology, as every smartphone has this Bluetooth functionality built in, every new technology comes in order to overcome problems of previous one. So, Bluetooth works in a limited range (distance). In order to get background knowledge of using this service I have created a chat app which works using Bluetooth, so you do not need an internet connection to communicate through this app. It asks for location permission and to enable Bluetooth. Then show your paired devices. If you start a scan, it shows all the available devices nearby and all the devices which are paired and available. If you select a device from available devices, it will pair that device. If you select an available device which is already paired, it will create a connection. Then you can send and receive messages from that device.
